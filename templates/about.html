Our Mission

The PneumoScan project aims to revolutionize early diagnosis of pneumonia using artificial intelligence. Leveraging the power of deep learning, we provide a tool to assist medical professionals by analyzing chest X-ray images with high accuracy (>92%) using the VGG16 model. Our goal is to reduce diagnostic time and improve patient outcomes globally.

Technology Stack

Machine Learning

  • VGG16 Architecture
  • TensorFlow 2.15.0
  • Keras Data Augmentation

Web Development

  • Flask 3.0.3
  • Tailwind CSS
  • Modern JavaScript

Dataset

Our project utilizes the Chest X-ray Pneumonia Dataset from Kaggle.

Training Set

5,216 images
(1,341 Normal, 3,875 Pneumonia)

Validation Set

16 images
(8 Normal, 8 Pneumonia)

Test Set

624 images
(234 Normal, 390 Pneumonia)
